# Chess / gothic chess variants

These new variants are just opening positions w/ a twist.
Normal rules apply.
In castling king is in the middle of rooks.
Castling in standard: "KQkq". Others use X-FEN.
Chess18, Chess324, FRC and DFRC are already invented.
Some variants are capped to 1000000, as they don't fit into RAM.
All files are in variants.zip
